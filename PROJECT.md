# Ultra – Open Accessibility Engine for Creative Software

## Overview

Ultra is an open-source accessibility engine designed to make complex creative software usable for visually impaired users.

It replaces visual-first interaction models with structured, non-visual workflows that can be operated through screen readers and keyboard-based commands.

## Problem

Modern creative tools such as video and audio editors are highly visual and difficult or impossible to use for blind and visually impaired users, even when partial accessibility features exist.

## Solution

Ultra introduces a new interaction model that transforms graphical interfaces into logical, structured workflows.

Instead of interacting with visual elements, users interact with contextual actions and commands that represent the same functionality in a non-visual format.

## Key Features

- Non-visual interaction model  
- Screen-reader optimized workflows  
- Structured command-based editing system  
- Reusable accessibility framework  

## Use Case

A visually impaired user can edit video or audio content without relying on visual timelines or graphical interfaces, using only screen reader feedback and logical commands.

## Status

- Prototype video editor developed  
- Accessibility concepts tested in real-world usage  
- Audio editor in development  

## Vision

To become a reusable accessibility layer that can be integrated into multiple software applications, improving accessibility across the creative software ecosystem.

## Author
